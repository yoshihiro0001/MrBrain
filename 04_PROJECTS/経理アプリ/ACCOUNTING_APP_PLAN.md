# 経理アプリ計画

## 目的
レシート・経費・カード明細・銀行明細・給与・税務申告を整理し、
申告時に苦労しない状態を作る。

## 最初に作る機能
1. 経費入力
2. レシート画像保存
3. OCR読み取り
4. 勘定科目候補
5. 月別集計
6. 法人別管理
7. 全力法人税に入れる数字の整理

## 将来作る機能
- カード明細CSV取り込み
- 楽天カード・オリコカード・JCB・みずほ銀行との連携補助
- 給与管理
- スタッフ別支払い管理
- 消費税集計
- 法人税申告補助
- e-Tax提出前チェック

## 重要
最初から完璧な自動ログインは目指さない。
まずはCSV取り込みと手入力補助で安定させる。

## 最小データ項目
| 項目 | 例 | 注意 |
|---|---|---|
| 日付 | 2026-06-22 | レシート日付 |
| 法人 | JOY / 孝行 | 個人支出と混ぜない |
| 金額 | 1100 | 税込・税抜を後で決める |
| 支払方法 | 現金 / カード / 銀行 | 明細連携用 |
| 勘定科目候補 | 消耗品費 | AI候補は確認する |
| 証憑 | ファイル場所 | 画像原本は安全に保管 |
