# Blueprint v2 Template

## 目的
Blueprint v2は、思想や発見を実装、運用、改善、自動化までつなげるための設計図テンプレートである。

今までのBlueprintに、完了条件、戻し方、運用方法、自動化判断を加える。

## 使うタイミング
- 経理を便利にしたい
- Booking.comを効率化したい
- ホテル業務を改善したい
- アプリや画面を作りたい
- Codexスキルを作りたい
- どこから相談すればよいか分からない
- 何度も同じ作業をしている

## 基本思想
いきなり実装しない。

まず、目的、現在地、完成形、接続する部品、テスト方法、戻し方をそろえる。

```md
思想
↓
Blueprint Engine
↓
Blueprint v2
↓
Blueprint Review
↓
承認
↓
最小実装
↓
運用
↓
Operation Log
↓
Evolution Loop
↓
Blueprint更新
↓
Automation Backlog
```

## Blueprint Engineチェック
Blueprint v2を書く前に、内部で次を確認する。

```md
1. 目的

3. 出力
4. 構成要素
5. 接続
6. 循環
7. 停滞
8. 最小介入
9. 将来の拡張性
10. 実行
```

## 注意
- Blueprintは長くするためのものではない
- 実装を遅らせるものではない
- 迷わず始めるための開始点である
- コードは最後に置く
- Blueprint Reviewを通過したものだけ実装へ進める
- できるだけ1枚で終わらせる
- 実装後は必ず `Operation Log` に結果を戻す
