## 現在の要点
- ホテルJOYを運営している
- 経理、スタッフ、設備、システム改善と接続する
- 予約返信、問い合わせ返信、クレーム対応は [[02_PRINCIPLES/Human Communication OS|Human Communication OS]] を参照する

## 接客・返信原則
- お客様は問題だけでなく感情を持って来館する
- 問題解決の前に理解と安心を置く
- Wi-Fi、設備不具合、予約問い合わせは「共感 → 確認 → 解決」の順番で対応する
- クレーム対応は「共感 → 謝罪 → 確認 → 解決」の順番で対応する
