# MrBrain

## 目的
MrBrainは、人生・会社・経理・税務・健康・契約・スタッフ管理・開発・思考整理を一元管理するObsidian Vaultです。

AIへ同じ説明を何度も繰り返さず、必要な背景を安全に渡せる状態を作ります。

## 使い方
- 最初に見る場所: [[00_HOME/HOME]]
- AIに渡す概要: [[07_SYSTEM/AI_CONTEXT]]
- 判断やルール: [[07_SYSTEM/AI憲法]]
- タスク管理: [[00_HOME/TASKS]]
- 重要な決定: [[07_SYSTEM/DECISION_LOG]]

## 入れるもの
- 事実、要約、判断基準、作業メモ
- 経理・税務・契約・スタッフ管理の索引
- 開発プロジェクトの計画と進捗
- AIに説明したい背景情報

## 入れてはいけないもの
- パスワード、認証コード、秘密鍵
- マイナンバー、銀行暗証番号、クレジットカード番号全文
- 契約書・申告書・給与データの原本そのもの
- 未確認情報を断定した文章

## 基本ルール
- 原本は安全な保管場所に置き、MrBrainには「場所・要約・確認日」を書く。
- 個人情報は最小限にする。
- フォルダを増やす前に、既存フォルダへ入れられないか確認する。
- AIへ渡すファイルは、見出しと箇条書きで整理する。
