# エネルギー循環OS

## 概要
エネルギー循環OSは、人や組織の行動から、エネルギーの流れを可視化する考え方。

目的は、評価や管理ではなく、停滞を発見し、循環を良くすること。

## 中核概念
- 行動は結果
- 欲求は原因
- 循環は構造
- 深層OSは、生存・つながり・拡張の3つ

## なぜ重要か
これまでの経理、JOY管理、孝行管理、スタッフ管理、相性AI、恋愛、採用、組織改善の構想は、別々のものに見えていた。

しかし、実際にはすべて「人・組織・社会のエネルギー循環を良くする」という一本の幹につながる。

## 既存プロジェクトとの関係
| 領域 | エネルギー循環OSでの意味 |
|---|---|
| MrBrain | 自分自身の循環を観測する実験機 |
| JOY | 組織と現場の循環を観測する場所 |
| 孝行 | 税務・経理・資産の停滞を減らす対象 |
| 経理 | お金の流れを見える化する循環ログ |
| スタッフ管理 | 人の役割とエネルギー供給源を見る場所 |
| 契約管理 | 固定費や責任の停滞を見つける場所 |
| 設備管理 | 現場の摩擦と改善余地を見る場所 |
| 相性AI | つながりの循環を良くする応用 |
| 株スクリーニング | 拡張と自由のための資本判断 |

## 重要な転換
以前:
```md
経理アプリを作る
```

現在:
```md
エネルギー循環OSの最初の実験機としてMrBrainを作る
```

## 保存すべき問い
- 何が停滞しているか。
- どこで摩擦が起きているか。
- 誰が悪いのではなく、どの循環が詰まっているか。
- どの行動ログから、どのエネルギーが推定できるか。
- 自分の生存・つながり・拡張は、今どの方向に動いているか。
